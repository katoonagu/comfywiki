# Как начать с ComfyUI

## Что нужно понять сначала

- ComfyUI удобнее всего осваивать через готовые workflow, а не с пустого холста.
- Для первых шагов важнее совместимость моделей, нод и VRAM, чем количество функций.
- Telegram-архив содержит много практических примеров, но wiki хранит только curated выжимку.

## Базовый путь

1. Понять setup и manager.
2. Научиться читать workflow JSON и его зависимости.
3. Освоить одну ветку генерации персонажей: portrait, turnaround или consistency.
4. Постепенно добавлять LoRA, custom nodes и upscale.
