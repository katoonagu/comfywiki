## Notes

- VNCCS (Visual Novel Character Creation Suite)** — новый инструмент для создания персонажей визуальных новелл**
- Многие пробовали генерировать арты с помощью нейросетей, но сталкивались с одной проблемой: как сделать так, чтобы персонаж оставался единым и узнаваемым на всех изображениях
- VNCCS — это мощный инструмент для создания спрайтов персонажей с единым стилем и внешностью. Теперь собрать полный набор артов для героя.
- **Этапы создания персонажа:**
- — Создание базового персонажа
- — Генерация наборов одежды
- — Генерация эмоций
- — Сборка готовых спрайтов
- — (Опционально) подготовка датасета для обучения LoRA
- Каждый шаг выполняется в отдельных workflows
- **Установка:**
- — Найдите VNCCS - Visual Novel Character Creation Suite в Custom Nodes Manager или установите вручную.
